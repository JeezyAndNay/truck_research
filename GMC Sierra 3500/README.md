# GMC Sierra 3500 HD — Duramax Diesel
*6.6L Duramax L5P V8 · SRW · Crew Cab · 4x4*

The Sierra 3500 runs the same Duramax/Allison platform as the Silverado 3500, but adds what the Silverado lacks: the **AT4 package** — the only factory off-road configuration available on a 1-ton GM truck. For buyers who need both heavy towing and genuine off-road capability, the Sierra 3500 AT4 is the strongest GM answer.

## Quick Stats (2024, SRW Crew Cab 4x4 Diesel)
- **Max tow:** 36,000 lbs (same as Silverado 3500)
- **Power/Torque:** 445 hp / 910 lb-ft (same as Silverado)
- **Transmission:** Allison 10L1000 10-speed
- **Injection pump:** CP4.2 (LML 2011–2016) ⚠️ / CP3 (L5P 2017+) ✅
- **Off-road package:** AT4 (available on 3500) ✅
