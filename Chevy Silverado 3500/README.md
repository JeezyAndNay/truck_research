# Chevy Silverado 3500 HD — Duramax Diesel
*6.6L Duramax L5P V8 · SRW · Crew Cab · 4x4*

The Silverado 3500 pairs the 6.6L Duramax V8 with the Allison 10-speed — the most praised transmission combination in the 1-ton class. The CP3 injection pump on all Duramax trucks makes this the safest used diesel buy in the segment from a fuel system perspective.

## Quick Stats (2024, SRW Crew Cab 4x4 Diesel)
- **Max tow:** 36,000 lbs (conventional)
- **Power/Torque:** 445 hp / 910 lb-ft
- **Transmission:** Allison 10L1000 10-speed
- **Injection pump:** CP3 (all Duramax years) ✅
- **Off-road package:** None dedicated on 3500
